# IMAGEAPI 

> Express api to upload images

## Build

```
npm i
```

or 

```
docker build -t imageapi:latest --force-rm .
```

## RUN

```
npm start
```

or

```
docker stack deploy -c stack.yml imagepai
```

## Stop

```
ctrl + 'c'
```

or 

```
docker stack rm imageapi
```
